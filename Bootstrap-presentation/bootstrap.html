Bootstrap

vad är Bootstrap?

  • Bootstrap är ett av de populärare projekten hos GitHub.
  • Det grundades av Mark Otto och Jacob Thornton på Twitter.
  • Består av HTML, CSS, JavaScript, Sass, LESS.
  • Kräver grundläggande kunskap HTML5.
  • Med Bootstrap kan du skapa snygg hemsida i responsiv design för mobil och surfplatta. Dvs. det finns hjälp och färdig kod för allt; till exempel menyer, navbar, knappar ,,,etc, som fungerar lika bra på alla stationära och mobila enheter.

Containers

För att applicera Bootstrap-layout på sitt dokument använder man speciella CSS-klasser. Alla Bootstrap-klasser ska finnas inuti en container-klass. Du kan ange någon av dessa två klasser nedan:


.container-klassen fyller sidan men har marginaler. Eller så använder man container för att koden skall placeras i mitten och ge lite marginaler på sidorna.

.container-fluid klassen fyller hela sidans bredd.

Grid system

Ett smart sätt att strukturera innehållet på sin hemsida är genom att göra kolumner. Bootstrap skapar CSS-klasser som vi kan använda för att beskriva kolumner. En sida delas in i 12 kolumner.
Du får gärna göra kolumner i olika storlekar, se bara till att de blir totalt 12 om alla ska få plats på en rad!

Vi vill skapa fyra stycken lika stora kolumner med olika innehåll. Om du tittar på koden så står det class=”col-sm-3” inom div-taggen. Vi vill att det skall bli siffran 12 med alla kolumner. Vi har fyra kolumner och alla med siffran 3. Det betyder att vi har 3 + 3 + 3 + 3 = 12 kolumners bredd på en liten skärm (Small, sm, Tablets (≥768px)).
